Public transit tools in ArcGIS Pro

Perform network analysis, visualize, and edit public transit data with out-of-the box tools in ArcGIS Pro.

ArcGIS Pro contains several out-of-the-box tools for working with public transit data.

As of ArcGIS Pro 2.4, the Network Analyst extension provides support for network analysis with public transit data. You can create network datasets with public transit data and use these to perform analyses for transportation and urban planning, public health, economic development, etc.

As of ArcGIS Pro 2.2, there are several tools for working with GTFS data in the Transit Feed (GTFS) toolset in the Conversion Tools toolbox. Use these tools to visualize transit stops and lines in the map and to edit and update your GTFS stops.txt file.
