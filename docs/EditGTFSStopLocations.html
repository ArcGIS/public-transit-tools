Edit GTFS Stop Locations

DEPRECATED. Use Edit GTFS Stop Locations to update your GTFS stops.txt file using ArcMap.

The Edit GTFS Stop Locations toolbox allows you to import your GTFS stops.txt file to ArcMap or ArcGIS Pro, use the ArcMap or ArcGIS Pro editing tools to update stop locations or attributes, and export the edits to an updated stops.txt file.

This tool is deprecated in ArcGIS Pro 2.2 and higher. To edit your GTFS stop locations in ArcGIS Pro 2.2 and higher, please use the GTFS Stops To Features and Features To GTFS Stops tools in the Conversion Tools toolbox.
